# Aislely 找货熊 — in-store product finder

Turn any grocery store into an instant, multilingual product finder. Staff snap a photo to
build the shelf map (zero typing); shoppers and new hires search in any language — even with
typos — and instantly see which aisle.

Built on Next.js 16 (MkSaaS base) · Better Auth · Aurora PostgreSQL (pgvector + tsvector + RLS) ·
AWS Bedrock (Nova vision + Titan embeddings) · deployed on Vercel.

- Live: https://aislely-stores.vercel.app
- Customer demo store: https://aislely-stores.vercel.app/s/demo
